# 建筑设计文档

> 状态：设计中（草稿）。标注"设计中"的条目尚未最终确定。

---

## 六棵建筑树总览

| 树 | 代号 | 职责 |
|---|---|---|
| 攻击塔 | A | 主动输出伤害，直接消灭敌人 |
| 地形塔 | T | 改变地图结构，引导路线 |
| 陷阱网 | M (Mesh) | 玩家手动布置，被动触发伤害 |
| 支援塔 | S (Support) | 增益友方塔 / 减益敌人 |
| 实验室 | L (Lab) | 全局经济、研究升级、场外支援 |
| 终极塔 | U (Ultimate) | 高成本超强特殊单体 |

---

## A 树——攻击塔

### 升级树结构

```
A1（基础炮塔）
├─ A21  单发强化（更强的单发伤害）
│   ├─ A31Aim        狙击炮
│   ├─ A31Bomb       加农炮
│   └─ A31Injection  病毒注射
├─ A22  连发加速（更快的攻速）
│   ├─ A32Lazer      激光
│   ├─ A32Lighting   闪电链
│   └─ A32Veteran    老兵
└─ A23  多向射击（同时打多个方向）
    ├─ A33Poison     毒素炮
    ├─ A33Mine       布雷塔
    └─ A33Fire       火焰炮
```

### 各炮塔说明

| 代号 | 名称 | 描述 |
|---|---|---|
| A1 | 基础炮塔 | 标准单发攻击，自动瞄准射程内最接近出口的敌人（first 策略，优先打快漏的） |
| A21 | 单发强化 | 伤害提升，攻速不变 |
| A22 | 连发加速 | 攻速大幅提升，单发伤害降低 |
| A23 | 多向射击 | 同时向多个方向发射，覆盖范围广 |
| A31Aim | 狙击炮 | 大射程、慢攻速、高单发；命中后给敌人挂"易伤"标记（受伤 +X%，命中刷新），先伤后标。索敌沿用最接近出口。手动蓄力选点留待以后 |
| A31Bomb | 加农炮 | 范围伤害，炮弹落点产生爆炸 |
| A31Injection | 病毒注射 | 命中后附加减速+持续伤害（DOT） |
| A32Lazer | 激光 | **持续光束**（非离散开火）锁定单一目标照射，dps=基础×斜坡倍率，持续打同一目标 N 秒升至 ×M；目标死亡/离开射程/换目标即重置斜坡。命中走统一管线吃易伤。专一单体强、对散兵弱（与闪电链互补） |
| A32Lighting | 闪电链 | 快攻速低单发；**瞬发电弧**（非飞行炮弹），从塔身连到首个目标，再向跳跃距离内最近的未命中敌人连锁跳跃，最多 N 跳、每跳伤害衰减。命中走统一伤害管线，故被标记敌人在链上自动吃放大。本版只传导伤害 |
| A32Veteran | 老兵 | 快攻速低基础；**最后一击**才算它的击杀，每击杀永久 +伤害且 +攻速（**伤害、攻速均不封顶**，攻速仅有引擎安全下限≈每帧一发），随击杀染金加深。整局有效、卖出清零。放主路核心位收益最大。成长叠在可调基础值之上 |
| A33Poison | 毒素炮 | 向前方扇形区域持续喷洒毒素 |
| A33Mine | 布雷塔 | 定时在射程内自动布置地雷；波次间隙可预先铺满雷区 |
| A33Fire | 火焰炮 | 小扇形火焰喷射，近距离高持续伤害 |

## T 树——地形塔

> 设计中

核心职责：占据格子改变地图结构，引导或限制敌人路线。与 A 树的区别在于不直接造成伤害。

---

## M 树——陷阱网 (Mesh)

> 设计中

核心职责：玩家手动在格子上放置静态陷阱，敌人经过时被动触发。
与 A33Mine 的区别：M 树由玩家主动操作放置，A33Mine 是塔的自动攻击行为。

---

## S 树——支援塔 (Support)

> 设计中

核心职责：增益周围友方塔（提升射速/伤害/射程）或对敌人施加减益（减速/弱化）。不直接造成伤害。

---

## L 树——实验室 (Lab)

> 设计中（部分方向已确定）

分三个子系统：

| 子系统 | 例子 | 机制 |
|---|---|---|
| 经济 | 铸币厂、矿场 | 占据格子，持续产出金币 |
| 研究 | 全塔攻速+5% | 购买一次，全场对应塔永久生效 |
| 场外支援 | 陨石、空袭、召唤援军 | 主动技能，有冷却时间；点击按钮后选择地图落点触发 |

场外支援触发方式：右栏点击按钮 → 地图上选取目标位置 → 触发效果。（参考王国保卫战陨石操作）
