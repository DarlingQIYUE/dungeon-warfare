# 建筑设计文档

> 状态：设计中（草稿）。标注"设计中"的条目尚未最终确定。

---

## 六棵建筑树总览

| 树 | 代号 | 职责 |
|---|---|---|
| 攻击塔 | A | 主动输出伤害，直接消灭敌人 |
| 地形塔 | T | 改变地图结构，引导路线 |
| 陷阱网 | M (Mesh) | 玩家手动布置，被动触发伤害 |
| 支援塔 | S (Support) | 增益友方塔 / 减益敌人 |
| 实验室 | L (Lab) | 全局经济、研究升级、场外支援 |
| 终极塔 | U (Ultimate) | 高成本超强特殊单体 |

---

## A 树——攻击塔

### 升级树结构

```
A1（基础炮塔）
├─ A21  单发强化（更强的单发伤害）
│   ├─ A31Aim        狙击炮
│   ├─ A31Bomb       加农炮
│   └─ A31Injection  病毒注射
├─ A22  连发加速（更快的攻速）
│   ├─ A32Lazer      激光
│   ├─ A32Lighting   闪电链
│   └─ A32Veteran    老兵
└─ A23  多向射击（同时打多个方向）
    ├─ A33Poison     毒素炮
    ├─ A33Mine       布雷塔
    └─ A33Fire       火焰炮
```

### 各炮塔说明

| 代号 | 名称 | 描述 |
|---|---|---|
| A1 | 基础炮塔 | 标准单发攻击，自动瞄准射程内最接近出口的敌人（first 策略，优先打快漏的） |
| A21 | 单发强化 | 伤害提升，攻速不变 |
| A22 | 连发加速 | 攻速大幅提升，单发伤害降低 |
| A23 | 多向射击 | 同时向多个方向发射，覆盖范围广 |
| A31Aim | 狙击炮 | 可蓄力手动选定目标，命中触发特殊效果（具体效果设计中） |
| A31Bomb | 加农炮 | 范围伤害，炮弹落点产生爆炸 |
| A31Injection | 病毒注射 | 命中后附加减速+持续伤害（DOT） |
| A32Lazer | 激光 | 对同一目标持续照射，秒伤随时间斜坡上升；脱靶后重置 |
| A32Lighting | 闪电链 | 伤害在周围敌人间连锁传导 |
| A32Veteran | 老兵 | 每击杀一个敌人永久叠加属性，越到后期越强；放在主路核心位收益最大 |
| A33Poison | 毒素炮 | 向前方扇形区域持续喷洒毒素 |
| A33Mine | 布雷塔 | 定时在射程内自动布置地雷；波次间隙可预先铺满雷区 |
| A33Fire | 火焰炮 | 小扇形火焰喷射，近距离高持续伤害 |

## T 树——地形塔

> 设计中

核心职责：占据格子改变地图结构，引导或限制敌人路线。与 A 树的区别在于不直接造成伤害。

---

## M 树——陷阱网 (Mesh)

> 设计中

核心职责：玩家手动在格子上放置静态陷阱，敌人经过时被动触发。
与 A33Mine 的区别：M 树由玩家主动操作放置，A33Mine 是塔的自动攻击行为。

---

## S 树——支援塔 (Support)

> 设计中

核心职责：增益周围友方塔（提升射速/伤害/射程）或对敌人施加减益（减速/弱化）。不直接造成伤害。

---

## L 树——实验室 (Lab)

> 设计中（部分方向已确定）

分三个子系统：

| 子系统 | 例子 | 机制 |
|---|---|---|
| 经济 | 铸币厂、矿场 | 占据格子，持续产出金币 |
| 研究 | 全塔攻速+5% | 购买一次，全场对应塔永久生效 |
| 场外支援 | 陨石、空袭、召唤援军 | 主动技能，有冷却时间；点击按钮后选择地图落点触发 |

场外支援触发方式：右栏点击按钮 → 地图上选取目标位置 → 触发效果。（参考王国保卫战陨石操作）
